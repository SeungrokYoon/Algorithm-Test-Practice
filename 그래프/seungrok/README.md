1. 매주 알고리즘 주제를 선정한다
2. 주제에 맞는 문제를 찾은뒤 문제와 소스코드를 남긴다

- 자기만의 branch를 생성한다
- '주제이름' 디렉토리 안에 '본인 이름' 디렉토리를 생성한다

3. 알고리즘 스크럼은 18:30에 진행한다
4. 매주 말에 main branch에 머지한다
